Customer-lifetime-value-estimation-using-purchase-frequency-and-average-order-value

Overview

This project focuses on estimating Customer Lifetime Value (CLV) using two key business metrics:
Purchase Frequency
Average Order Value (AOV)
CLV helps businesses understand how much revenue a customer is expected to generate over time, enabling better decisions in marketing, retention, and customer segmentation.

Objective

To build a simple and effective model that calculates customer value based on historical transaction data and provides actionable insights.

CLV Formula

We use a basic and widely accepted formula:
CLV=Purchase Frequency×Average Order Value×Customer Lifespan
Where:
Purchase Frequency = Total Orders / Total Customers
Average Order Value (AOV) = Total Revenue / Total Orders
Customer Lifespan = Average duration a customer continues purchasing

Use Cases

Customer segmentation (high-value vs low-value)
Targeted marketing campaigns
Retention strategies
Revenue forecasting
